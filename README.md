# nextjs-app-router-authjs-graphql-nexus

Next.js(App Router) boilerplate.

## Stack

* TypeScript
* Next.js
* Auth.js(NextAuth.js
* Primsa
* MySQL
* GraphQL
* Nexus
* Apollo Client
* Apollo Server

## Usage

### Install 

```
npm i
```

### Set up

```
// prisma
npx prisma db push

// graphql schema
npm run build:nexus-typegen
```
